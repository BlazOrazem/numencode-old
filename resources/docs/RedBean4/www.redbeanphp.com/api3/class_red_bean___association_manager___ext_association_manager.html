extAssociateSimple (   $beans1,
  $beans2,
  $extra = NULL 
)

Simplified version of extAssociate(). Associates two beans $bean1 and $bean2 with additional properties defined in third parameter $extra. This third parameter can be either an array, a JSON string, a single value (will be assigned to property 'extra') or a bean.

This method has been deprecated, please use $bean->link instead to form many-to-many associations with additional properties.
